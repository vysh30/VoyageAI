# 🌍 VoyageAI

AI-powered travel planning system built with Google Opal.

VoyageAI transforms a destination into a complete travel experience by generating:

* 📍 Trending attractions
* ☀️ Weather analysis
* 🗓️ Personalized 3-day itinerary
* 🗺️ Google Maps recommendations
* 🌐 Mobile-friendly travel webpage

---

## 🚀 Overview

Planning a trip often requires searching across multiple websites for attractions, weather updates, maps, and itinerary suggestions.

VoyageAI automates this process through an AI workflow that generates a complete travel plan in minutes.

---

## ⚙️ Workflow Architecture

Destination Input
↓
Trending Spots Finder
↓
Weather Analyzer
↓
3-Day Travel Planner
↓
Travel Webpage Builder
↓
Travel Planner Output

---

## ✨ Features

### 📍 Trending Destination Discovery

Identifies popular attractions, landmarks, and local experiences.

### ☀️ Weather Intelligence

Provides weather forecasts and travel recommendations.

### 🗓️ Smart Itinerary Generation

Creates structured day-by-day travel plans.

### 🗺️ Google Maps Integration

Provides location references and navigation support.

### 🌐 Dynamic Travel Webpage

Automatically converts itineraries into a modern travel webpage.
